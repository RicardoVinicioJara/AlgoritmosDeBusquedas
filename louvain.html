Louvain

Es un método simple, eficiente y fácil de implementar para identificar comunidades en redes grandes.
El algoritmo Louvain es un algoritmo jerárquico de agrupación en clústeres, que combina comunidades de forma recursiva en un único nodo y ejecuta la agrupación en clústeres de modularidad en los gráficos condensados.
El análisis de una red típica de 2 millones de nodos toma 2 minutos en un PC estándar. El método revela las jerarquías de las comunidades y permite hacer zoom dentro de las comunidades para descubrir subcomisiones

Uso

El método Louvain se ha utilizado para investigar el cerebro humano y encontrar estructuras jerárquicas de la comunidad dentro de la red funcional del cerebro. El estudio mencionado es Modularidad Jerárquica en Redes Funcionales del Cerebro Humano.

Ver Comunidades

A continuación, ejecutaremos un algoritmo que separara en comunidades a los nodos, y de estas nos indicara cuales podrían ser sus otras comunidades, o dicho de otra manera a que comunidad podrían pertenecer. Cade hay que destacar que para este algoritmo se tubo que modifica la base de datos agregando mas conexiones de las que eran necesarias ya que así se podía encontrar mas comunidades.

CALL gds.louvain.stream({ nodeProjection: 'Node', relationshipProjection: { TYPE: { type: 'TYPE', orientation: 'undirected', aggregation: 'NONE' } }, includeIntermediateCommunities: true }) YIELD nodeId, communityId, intermediateCommunityIds RETURN gds.util.asNode(nodeId).name AS Nodo, communityId as ID_Comunidad, intermediateCommunityIds as Comunidades ORDER BY Nodo ASC
